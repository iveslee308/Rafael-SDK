Detailed Description

Structure for status read/write.

require_mode is bitwise read/write operation. For example, if reqquired_mode = (FLASH_STATUS_RW1 | FLASH_STATUS_RW2) then the flash driver will read/write status1 register and status2 register, depends on read/write API.
