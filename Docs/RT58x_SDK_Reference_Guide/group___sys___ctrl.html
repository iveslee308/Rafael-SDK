◆ sys_set_retention_reg()

__STATIC_INLINE void sys_set_retention_reg ( uint32_t  index,
uint32_t  value 
)

sys_set_retention_reg. Use to save some retention value.

Parameters
indexThe index for which scratchpad register to save It should be 0~7.
valueregister value

Please notice when system power-reset (cold boot), all system retention scratchpad register (0~7) will be reset to zero. System scratchpad register 6 is used for system hardware control: BIT0 for enable resetting CM3 peripherals when watchdog reset is triggered. BIT1 for enable resetting APBGPIO and Remap when watchdog reset is triggered. BIT2 for enable resetting communication subsystem when watchdog reset is triggered.

System scratchpad register 7 is used for system hardware control: BIT0 for skip ISP function when system wakeup from deep sleep. This function is very useful for real product. BIT1 for RT58x communication subsystem used. BIT2 for RT58x application project to notify wakeup from sleep.

So users can use system retention scratchpad register 0~5 for their special purpose.
