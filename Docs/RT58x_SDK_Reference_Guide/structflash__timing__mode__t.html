Detailed Description

Flash config structure holding flash timing configuration for flash suspend/resume/deep-powerdown.

Please read Flash Datasheet for suspend/resume/powerdown timing. The unit of this data member is in HCLK clock, for example, if CPU running in 32MHz, and flash deep power down is 3 us, then deep_pd_timing is 3*32, since 32 clock is 1 us.
