Composes a string with the same text that would be printed if format was used on printf, but instead of being printed, the content is stored as a C string in the buffer pointed by pbuf.

Parameters
[in]pbufPointer to a buffer where the resulting C-string is stored. The buffer should have a size of at least buf_size characters.
[in]buf_sizeMaximum number of bytes to be used in the buffer. The generated string has a length of at most buf_size - 1, leaving space for the additional terminating null character.
[in]fmtC string that contains a format string that follows the same specifications as format in printf
[in]...Depending on the format string
Returns
